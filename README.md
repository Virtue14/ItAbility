## 개발자 소개

- 🍊김규린: 게시글 및 댓글 구현
- 🍎박경덕: 로그인 기능 구현
- 🍐박고은: 모집글 및 신청, 관리 구현
- 🍉서승엽: 신고, 블랙리스트, 팔로우 기능 구현
- 🍓이현우; 마이패이지 기능 구현
